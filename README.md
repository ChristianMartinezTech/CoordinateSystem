# CoordinateSystem

C# console program that takes the X Y coordinate values of the 3 corners of a triangle and calculates:
- Length of triangle sides
- Is it equilateral?
- Is it isosceles?
- Is it right?
- Perimeter
- Even numbers from 0 to value of triangle perimeter
